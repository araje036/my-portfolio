# My Portfolio

This repository contains a simple single-page portfolio website built with HTML and CSS.

## Structure
- `index.html` — main portfolio page
- `style.css` — styling for the site
- `images/` — put your profile image here as `profile.jpg`

## Preview locally
1. Open `index.html` directly in your browser.
2. Or use a local server, for example:
   - `python3 -m http.server 8000`
   - open `http://localhost:8000`

## Add your profile image
- Place a square or portrait image in `images/profile.jpg`
- Recommended size: `800x800` for a crisp display
- The site will render it at `160x160`

## Publish with GitHub Pages
1. Go to your repository on GitHub.
2. Open `Settings` → `Pages`.
3. Select branch `main` and folder `/ (root)`.
4. Save and wait a minute.
5. Your site should appear at `https://araje036.github.io/my-portfolio/`

## Notes
- The page uses Font Awesome via CDN for icons.
- Customize text, colors, and project cards in `index.html`.
